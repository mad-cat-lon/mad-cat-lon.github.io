projects

jargonaut

A Python 3.9 source-to-source obfuscation engine with a focus on being extensible, configurable and easy to use. Implemented features include mixed boolean-arithmetic expressions of various classes, variable insertion to prevent lifting, bytecode patching at runtime, bogus control flow, opaque predicates and much more. Parsing and manipulation of source code as an AST is done with LibCST and Pyre is used for type inference during code generation and analysis. Most math is done with Z3

visuallama

Simple MERN stack webapp that allows you to interact with locally running LLMs and manage parameters such as temperature, starting prompt, max tokens, etc.
